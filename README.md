# Categories

A web app for creating and sharing custom word puzzles with friends. Build a puzzle with 4 groups of 4 related words, generate a shareable link, and challenge others to find the connections.

## Play

Visit [matthewtwarren.github.io/categories](https://matthewtwarren.github.io/categories/) to create or play a puzzle.

## Features

- Create puzzles with 4 color-coded groups (yellow, green, blue, purple)
- Share puzzles via URL - no account needed
- Fixed 4Ã—4 board that doesn't reflow when groups are solved
- "One away" feedback when 3 of 4 selections match a group
- Reveal answers after failed attempts

## Run locally

```bash
npm install
npm run dev
```

## Tech stack

React, TypeScript, Vite, Tailwind CSS
